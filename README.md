# 测试日常刚需工具

> 输入业务接口描述 / 接口 JSON，自动生成正向、反向、边界、异常、场景测试用例，一键导出 Excel

## ✨ 特性
- 自动生成多维度测试用例（正向、反向、边界、异常、业务场景）
- 支持接口描述或 JSON 输入
- 一键导出 Excel 文件，便于评审与执行

## 🚀 使用步骤

### 1. 填入自己的 key
在 ai_api.py中设置你的 API key且可更改你的模型或修改对应的api文件。

### 2. 运行 main.py

python main.py


### 3. 在窗口处输入接口信息

按照提示依次填写接口相关信息，示例格式如下：


接口名称：用户登录
请求方式：POST
接口地址：/api/login
入参：username(字符串，必填)、password(字符串，必填，6-16位)
业务描述：用户输入账号密码登录，成功返回token，失败返回对应提示


### 4. 生成结果示例





导出 Excel 后可直接用于测试执行与管理。
