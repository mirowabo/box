测试日常刚需工具：输入业务接口描述 / 接口 JSON，自动生成正向、反向、边界、异常、场景测试用例，一键导出 Excel
使用步骤
1.填入自己的key
2.运行main.py
3.在窗口处输入接口信息，
例如：
接口名称：用户登录
请求方式：POST
接口地址：/api/login
入参：username(字符串，必填)、password(字符串，必填，6-16位)
业务描述：用户输入账号密码登录，成功返回token，失败返回对应提示
